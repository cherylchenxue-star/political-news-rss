# 全球政治头条 RSS

每天早上自动聚合新华网、中新网、新浪新闻等权威媒体的头条政治新闻，智能筛选热度最高的内容。

## 功能特性

- **多源聚合**：新浪国内/国际 API、中新网 RSS、新华网网页抓取
- **智能标签**：自动识别 14 类政治标签（中美关系、台海局势、俄乌冲突等）
- **热度排序**：基于来源权重 + 关键词 + 时间衰减的综合算法
- **RSS 订阅**：标准 RSS 2.0 格式
- **精美前端**：响应式网页，支持分类筛选、搜索、标签云

## 数据源

| 来源 | 类型 | 内容 |
|------|------|------|
| 新浪新闻 | API | 国内/国际头条 |
| 中新网 | RSS | 国内/国际/滚动 |
| 新华网 | 网页 | 时政/国际 |

## 部署

本仓库通过 GitHub Actions 自动部署到 GitHub Pages：

- 每天北京时间 **07:00、12:00、19:00** 自动抓取
- 推送代码时自动触发构建

## 访问

- **前端页面**：`https://你的用户名.github.io/political-news-rss/`
- **RSS 订阅**：`https://你的用户名.github.io/political-news-rss/political_news_rss.xml`

## 本地运行

```bash
pip install -r requirements.txt
python fetcher.py
```

然后打开 `index.html` 查看前端页面。
